## 关于FIDIS

FIDIS（全称Flexible information data integrate system，柔性信息与数据集成系统），作为智物联工业物联网体系中的应用系统平台，承担着MixIOT应用数据的展示，以及与MixIOT系统的操作和交互。

同时，FIDIS系统作为底层应用平台，支持基于OpenFrame开发框架的应用的二次开发。
